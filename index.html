Frequently Asked Questions

1.The difference between var, let, and const
Var is an old variable system,it can do redeclare,hoisted,function-scope.on the other hand,let can do reassign (value can change)not redeclare.its not alowed for hoisting.We can use const when value will fixed(reassign,redeclar is not allowed for const).
2.the difference between map(), forEach(), and filter()
"map()"is a builtin function.it works on each element and do the task which is given in callback function and returns an array.forEach() does the same thing but its not return array.filter() also does the same thing but it works on conditon and returns array.Notice that all the builtin function is like a loop and it works on each elements of array.
3.explain arrow functions and how they are different from regular functions
Regular function is a JavaScripts old way but Arrow function is a ES6 function.this behavior is allowed for regular function not for arrow function.at the same wAy,hoisting is allowed for only regular function.
4.how JavaScript Promises work
Promise is an object of javascript which represent that given task will finish in future.javascript has 3 states--Pending,resolve,reject.
5.how closures work in JavaScript
Inner function is remembering his outer or parent functions variable and its called closure.
